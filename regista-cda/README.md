# 🏆 REGISTA - Application de Gestion (CDA)

Bienvenue sur le dépôt de Regista, une application développée dans le cadre de mon diplôme de Concepteur Développeur d'Applications (CDA).

## 🚀 Technologies utilisées
* **Frontend :** React.js (Vite), TailwindCSS / Material UI
* **Backend :** Node.js, Express
* **Base de données :** MySQL / PostgreSQL (à venir)
